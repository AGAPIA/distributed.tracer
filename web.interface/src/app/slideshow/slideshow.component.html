Welcome

Cardinal is a collaborative, scalable, deep program inspection framework based on the River intermediate language.

Keep scrolling for additional framework details.

Deep program inspection

Cardinal combines features of fuzzers and symbolic execution in order to generate a test corpus, with the goal of maximizing code coverage.

The framework accepts x86 Windows (PE) and Linux (ELF32) binaries, and optionally, a static test corpus or a test generator.

Scalable

Cardinal is composed on several loosely coupled components that can be instanced as many times as necessary to cover available computing resources.

Collaborative

Cardinal uses multiple testcase sources

  • The provided corpus
  • The provided generator
  • Orthrus, for generating an initial testcase corpus
  • A testcase generator based on genetic algorithms
  • Heuristic testcase generators based on the AFLfast fuzzer
  • The River symbolic execution framework
